# AeroMix

## Setting up Ableton

1. Create new MIDI track.
1. Import music into the MIDI track.

### Volume (thumb)

1. Import utility into the MIDI track.
1. Go to MIDI mapping mode, select gain, and curl thumb.

### Reverb/echo (index finger)

1. Import reverb into the MIDI track.
1. Go to MIDI mapping mode, select decay, and curl index finger.
